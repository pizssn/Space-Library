# Space Libray——星空图书馆
这是小白的第一个springboot项目，是一个十分简单的前后端分离的项目，主要采用 Vue.js + SpringBoot 技术栈开发。当然我只负责的后端。

------

## 主要技术栈
#### 前端
1.Vue.js
2.ElementUI
3.axios

### 后端
1.Spring Boot
2.Apache Shiro 3.Apache Log4j2 4.Spring Data JPA

### 数据库
MySQL
